# Git Commands Reference

| Command                             | What it does                                                              |
|-------------------------------------|---------------------------------------------------------------------------|
| `git init`                          | Initialize a new git repository in the current folder                     |
| `git status`                        | Show what files changed, what's staged, what's untracked                  |
| `git add .`                         | Stage all changed files in the current directory for commit               |
| `git add <file>`                    | Stage a specific file only                                                |
| `git commit -m "message"`           | Save a snapshot of staged files with a label                              |
| `git log`                           | Show commit history                                                       |
| `git remote add origin <url>`       | Connect local repo to a remote GitHub repository                          |
| `git push -u origin master`         | Push commits to GitHub and set upstream tracking                          |
| `git push`                          | Push new commits to GitHub (after upstream is set)                        |
